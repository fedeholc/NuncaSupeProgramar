### Instalación

```bash
npm install --save-dev jest
```

Para usar con TypeScript instalar también @types/jest:

```bash
npm install --save-dev @types/jest
```

### Correr test

**Una vez:**

```bash
npx jest
```

**En modo watch:**

```bash
npx jest --watchAll
```
